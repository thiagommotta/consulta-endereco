# consulta-endereco
API para consulta de endereço de acordo com o CEP informado.

É consultado o endereço completo de um logradouro de acordo com dados fornecidos pelo Via Cep

Além disso, é calculado um valor de frete para cada estado, de acordo com a região onde este estado está localizado.

| Região       | Frete    |
|--------------|----------|
| Norte        | R$ 20,83 |
| Nordeste     | R$ 15,98 |
| Centro-Oeste | R$ 12,50 |
| Sudeste      | R$ 7,85  |
| Sul          | R$ 17,30 |
